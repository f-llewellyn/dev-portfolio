PROJECTS

read.me — Write a readme like a pro!

Summary:

Read.me is a web app that enables users to create awesome READMEs for their open-source projects with an editor, live preview and pre-made markdown elements. Users can also pull existing READMEs from their GitHub repositories and edit them in the web app, then download it locally to add to their next commit.

Challenges:

  • The biggest challenge with this project was integrating with the GitHub rest API, as the documentation isn’t amazing. Despite this, I was eventually able to integrate it into the application. Due to the lack of resources on using the GitHub API with sveltekit, I wrote this article on DEV.to for others to view if they ever come across the same issue.
  • Another challenge I faced with this project was making it responsive for mobile devices, as there’s a lot of content and interactive elements to fit on such a small screen.

Live Preview: https://read-me-murex.vercel.app/

GitHub Repository: https://github.com/f-llewellyn/read.me/

Technologies: SvelteKit, Tailwind, JavaScript, CSS, Vercel, Figma

JobFair — Don’t get mugged off!

Summary:

JobFair is a prograssive web app (PWA) that aims to allow young people to find reliable and secure employment without fear of being taken advantage of.

Users are able to search for jobs by filtering through pay, hours, location and title in real time.

Challenges:

  • The real time job filter was somewhat challenging as it required a lot of state management to work effectively. Luckily, I was using Vue 3, so I was able to use the composition API which made it far easier, and allowed me to avoid prop-drilling
  • Implementing the PWA features was also difficult as I was using a library for Vite that had very little in the way of documentation, so I had to go through troubleshooting myself. Eventually, I did get it to work through some additional configuration.

Live Preview: https://jobfair-fl.herokuapp.com/

GitHub Repository: https://github.com/f-llewellyn/jobfair/

Technologies: Vue, MongoDB, Node.js, Express, CSS, JavaScript, PWA, Vite, Figma
